# Claude Token Saver AI 🚀

A lightweight Chrome Extension that uses **Gemini 1.5 Flash AI** to smartly compress your prompts before sending them to Claude.ai, helping you save tokens and extend your conversation limits.

---

## ✨ Features
- **Smart Compression:** Removes redundant words while keeping technical terms and constraints intact.
- **Privacy First:** Your Gemini API Key is stored locally in your browser.
- **Seamless Integration:** Adds a "Smart Compress" button directly into the Claude.ai input area.

## 🛠️ Installation (Manual)
Since this extension is not yet on the Chrome Web Store, follow these steps to install it:

1. **Download the code:** 
   - Click the green `Code` button and select `Download ZIP`.
   - Extract the ZIP file to a folder on your computer.
2. **Open Chrome Extensions:** 
   - Go to `chrome://extensions/` in your browser.
3. **Enable Developer Mode:** 
   - Toggle the **Developer mode** switch in the top-right corner.
4. **Load the Extension:** 
   - Click **Load unpacked** and select the folder where you extracted the files.

## 🔑 How to set up the API Key
1. Go to [Google AI Studio](https://aistudio.google.com/).
2. Create a free API Key for **Gemini 1.5 Flash**.
3. Click the extension icon in your browser, paste your key, and click **Save**.

## 🚀 How to Use
1. Open [Claude.ai](https://claude.ai).
2. Write your prompt in the input box.
3. Click the **✨ ضغط ذكي (AI)** button.
4. Your prompt will be instantly compressed! Now press Enter to send.
